üìã What is PostHog?

PostHog is an open-source product analytics platform that helps you understand user behavior, run A/B tests, and build better products.

Features include:

  • Product Analytics
  • Session Recording
  • Feature Flags
  • A/B Testing
  • User Surveys
  • And much more...

Option 1: PostHog Cloud (Recommended)

The easiest way to use PostHog is through their hosted cloud service:

Sign Up for PostHog Cloud

Option 2: Self-Hosted Deployment

For full control, deploy PostHog on your own infrastructure:

  • Docker Compose: Quick start for development and testing
  • Kubernetes/Helm: Production-ready deployment with scaling
  • Cloud Platforms: AWS, GCP, Digital Ocean, etc.

Option 3: Hobby Deployment

Run PostHog locally for learning and experimentation:

docker run -d --name posthog -p 8000:8000 posthog/posthog:latest
